# Vehicles Market Dashboard

## Descripción

Esta aplicación web fue desarrollada con Streamlit para visualizar datos de anuncios de venta de vehículos en Estados Unidos.

La aplicación permite:

- Visualizar la distribución de precios mediante un histograma.
- Analizar la relación entre el año del modelo y el precio con un gráfico de dispersión.
- Explorar los datos de forma interactiva.

## Tecnologías utilizadas

- Python
- Pandas
- Plotly
- Streamlit

## Cómo ejecutar la aplicación localmente

1. Crear y activar el entorno virtual.
2. Instalar dependencias:

pip install -r requirements.txt

3. Ejecutar la aplicación:

streamlit run app.py
